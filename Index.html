Hoje nós iremos falar um pouquinho sobre a linguagem de programação mais difícil e mais complexa já vista no mundo da programação...sim, estamos falando dele mesmo, o HTML :3 (brinks)

O HTML é uma sigla para Hyper Text Markup Language, que em português significa "Linguagem de Marcação de Hipertexto", que nada mais é do que a linguagem de marcação utilizada para a construção de páginas na Web, incluse esta foi inteiramente escrita apenas em HTML. Legal, né?

Como você pode ver, essa página não é tão bonita quanto a maioria dos sites, blogs e afins que estamos acostumados a ver por aí, e isso porque o HTML é apenas o "esqueleto" de uma página. Para construirmos páginas mais bonitas precisamos da ajuda de um mecanismo para adicionar estilo ao documento web, um cara chamado CSS.
