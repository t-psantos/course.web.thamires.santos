Guia de HTML Básico para iniciantes

E aí galera, beleza?

Hoje nós iremos falar um pouquinho sobre a linguagem de programação mais difícil e mais complexa já vista no mundo da programação: sim, estamos falando dele mesmo... o HTML (brinks)

O HTML é uma sigla para Hyper Text Markup Language, que em português significa "Linguagem de Marcação de Hipertexto", que nada mais é do que a linguagem de marcação utilizada para a construção de páginas na Web. Uma página escrita somente em HTML não é tão bonita quanto a maioria dos sites, blogs e afins que estamos acostumados a ver por aí e isso porque o HTML é apenas o "esqueleto" de uma página.

Para construirmos páginas mais bonitas precisamos da ajuda de um mecanismo para adicionar estilo ao documento web, um cara chamado CSS. Mas calma que esse é um assunto para outro dia..."vamos por partes" como diz Jack, o Estripador rsrs

Tags

As Tags, no HTML, são responsáveis por denotar uma marcação e quase sempre elas tem um início e um fim para que o navegador possa renderizar a página. Elas são representadas pelos sinais de < (menor que) e > (maior que).

<span>

É um elemento genérico que pode ser usado para agrupar elementos para fins de estilo(utilizando os atributos "class" e "id"). Ele deve ser utilizado em linha.

<div>

Também trata-se de um elemento genérico, entretanto <div> é um elemento de bloco, utilizado para realizar divisões na página.
Não se preocupe, esses conceitos ficarão mais claros quando falarmos de CSS.

Listas

No HTML, também é possível utilizar listas. Elas podem ser ordenadas ou não-ordenadas e são representadas pelas tags <ol> e <ul>, respectivamente. Cada item da lista é definido pela tag <li>.



As listas possuem atributos como o "start", que especifica o valor inicial para a ordem dos itens, o "reversed", que lista do menos importante ao mais importante, e o "type" que indica o tipo da numeração. Os tipos de numeração são:
  • 'A'
  • 'a'
  • 'i
  • 'I'
  • e '1' (padrão)

Tabelas

Em HTML, as tabelas são definidas pela tag <table> e uma célula de dados pode conter texto, imagens, listas, parágrafos, formulários, tabelas, etc.

Tag Descrição
<table> Define uma tabela
<th> Define um cabeçalho de uma tabela
<tr> Define uma linha de tabela
<td> Define uma célula de tabela
<caption> Define um título
<thead> Define um cabeçalho
<tbody> Define o corpo de uma tabela
<tfoot> Define o rodapé(footer) de uma tabela

Formulários

Os formulários são amplamente utilizados em páginas web, normalmente para coletar dados ou informações do usuário seja para autenticação, cadastro, edição de dados etc. A tag <form> demarca um formulário e o atributo <action=""> define o destino dos dados que serão inseridos pelo usuário.

Input

O <input> é a tag utilizada para a entrada de dados e possui atributos como: readonly, disabled, autofocus, multiple, entre outros.
Normalmente os inputs possuem um tipo:

Categoria Type Descrição Visual
Texto text Input de texto(normalmente o mais utilizado)
password Esconde o texto(utilizado normalmente para senhas)
email Facilita a validação de emails e conta com preenchimento automático
search Facilita integrações com mecanismos de buscas
url permite a inserção de uma URL (facilita validação e sugestões do browser)
Datas date campo de data com calendário
datetime-local campo de data com calendário e horário
month campo para seleção de um mês
time permite selecionar um horário
week seleciona uma semana(pelo número dela)
Números range seleção de range (volume, tamanho, quantidade, etc)
number seleção de números
Seleção select opções de seleção pré-definidos
checkbox multiseleção de opções
radio seleção única
Outros color campo de seleção de cor em Hex ou RGB
file envio de arquivos para servidor, como upload de fotos, etc
Botões submit faz o envio de todos os dados do formulário que estiver dentro da tag <form>
button cria um botão genérico
reset limpa todos os campos preenchidos

⚠ Importante! ⚠
Apesar de ser comum ver os inputs do tipo "button", é uma boa prática utilizar a tag <button></button> caso você queira adicionar botões em sua página. Isso porque dessa forma os mecanismos de busca conseguem "rankear" melhor a sua página.

Mídias

Muitas vezes, dependendo do conteúdo que você deseja apresentar em sua página, se faz necessário o uso de mídias (vídeos e áudios, por exemplo) e por isso é importante saber como utilizá-las.

Áudios

No geral, para adicionar áudio utilizamos a tag <audio> onde "controls" mostra os controles e "type" define o tipo.



O código ficaria assim:

<audio controls>
   <source src="caminho_do_arquivo_de_audio" type="tipo/formato">
   <source src="caminho_do_arquivo_de_audio" type="tipo/outroformato">
   Seu browser não suporta este formato de áudio!
<audio>
